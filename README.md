## 🚀 Quick Start

### **Prerequisites**
- Node.js 18+ 
- npm 8+
- Git

### **Installation**
```bash
# Clone the repository
git clone https://github.com/helm-ai/helm-ai.git
cd helm-ai

# Install dependencies
npm install

# Copy environment configuration
cp .env.example .env

# Start the server
npm run dev
```

### **Docker Setup**
```bash
# Build the image
docker build -t helm-ai .

# Run the container
docker run -p 3001:3001 helm-ai
```

## 🏗️ Architecture

### **Project Structure**
```
helm-ai/
├── src/
│   ├── ai/
│   │   ├── core/           # Core AI capabilities
│   │   ├── security/       # Security and safety systems
│   │   └── ip/             # IP protection and analysis
│   ├── api/
│   │   ├── routes/         # API route definitions
│   │   ├── middleware/     # Express middleware
│   │   └── controllers/    # Request controllers
│   └── config/             # Configuration files
├── docs/                   # Documentation
├── tests/                  # Test suites
├── server.js              # Main server file
└── package.json           # Dependencies and scripts
```

### **Technology Stack**
- **Backend**: Node.js, Express.js
- **Security**: Helmet, JWT, bcrypt
- **Database**: SQLite (development), PostgreSQL (production)
- **Monitoring**: Winston, Morgan
- **Testing**: Jest, Supertest
- **Documentation**: JSDoc
